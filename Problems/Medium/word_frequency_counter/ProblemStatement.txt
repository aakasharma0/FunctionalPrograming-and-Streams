Problem:

Input:

java = 3,
streams = 1
}

Rules:

- Case insensitive
- Ignore extra spaces
- Do not use loops
- Use Java 8 Stream API only

Concepts Tested:

- split
- map
- groupingBy
- counting
- String normalization
